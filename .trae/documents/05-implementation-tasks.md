## 用户使用流程（完整版）

### 离线阶段（一次性）

```bash
# 1. 拍摄所有拼图块（平铺在桌面上，手机正上方拍摄）
# 2. 拍摄包装盒上的完整图
# 3. 运行预处理

python src/run_batch.py \
  --photos-dir ./my_puzzle/photos \
  --target ./my_puzzle/box_image.jpg \
  --width 20 \
  --height 15 \
  --output-dir ./my_puzzle/output

# 4. 查看结果
#    output/piece_catalog.html     → 编号对照
#    output/annotated_target.png   → 标注目标图
#    output/solution_grid.txt      → 文字解方案
#    output/database/              → 数据库文件（在线阶段使用）
```

### 在线阶段（拼图时反复使用）

```bash
# 拼图过程中，拍照当前手中的拼图块

python src/run_realtime.py \
  --database ./my_puzzle/output/database \
  --photo ./current_piece.jpg

# 输出：
#   "拼图块 #42 → 放在第 3 行第 7 列，不需要旋转"
#   "新拼图块 #301 → 放在第 12 行第 5 列，顺时针旋转 90°"
#   "新拼图块 #302 → 位置未知，已添加到数据库（需要更多块才能确定）"
```
